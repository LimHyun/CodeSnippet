Javascript
==========

작업중 간간히 사용하는 JS코드들로 jQuery 1.7.x를 활용한다.

*향후작업예정
1. 아래에 기재된 소스 각각을 클래스화(필요한 곳에서만 쓸 수 있도록)
2. 클래스화 된것들을 묶어서 하나의 단독 클래스로 제작(2개 이상 사용할 시 통합관리)
3. 2번을 하기 위해 동적로딩 방식으로 변경
4. 불필요한 코드 제거 / 개선


SelectTab

AutoYoutubeTag
